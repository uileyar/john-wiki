Claude Code Pro/Max订阅默认只是中档大脑    
\*\*Pro / Max 订阅用户，Claude Code 默认给的 effortLevel 是 \`medium\`；走 API 的用户默认才是 \`high\`。\*\*  
如果你也是 Pro / Max 订阅用户，真心建议花 30 秒打开 settings.json 看一眼自己现在的档位。
