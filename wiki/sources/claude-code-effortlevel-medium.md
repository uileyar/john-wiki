## 摘要

这条短文提醒 [[ClaudeCode]] Pro/Max 订阅用户检查自己的 [[effortLevel]] 设置：文章声称订阅用户默认是 `medium`，而通过 API 使用时默认才是 `high`。它的实际价值不是展开工作流，而是把“模型能力档位”作为 [[ClaudeCode]] 配置检查项。

## 关键论点

- [[effortLevel]] 会影响 [[ClaudeCode]] 的推理强度与响应方式。
- Pro/Max 订阅用户不应默认假设自己正在使用最高档位。
- 最短行动是打开 `settings.json` 查看当前配置。
