# 三层架构

[[三层架构]] 是 [[持久化Wiki]] 的基础组织方式：`raw/` 保存不可变源，`wiki/` 保存 LLM 维护的结构化页面，`CLAUDE.md`/`AGENTS.md` 保存 schema 与工作流约束。

## 要点

- `raw/` 是来源层，保存文章、论文、转录、草稿等原始材料。
- `wiki/` 是知识层，包含 source、entity、concept、synthesis、overview、index 和 log。
- schema 文件是治理层，告诉 agent 如何摄入、查询、体检、建图和命名。

## 关联

- [[持久化Wiki]] —— 三层架构服务的核心范式。
- [[知识复利]] —— 三层架构让每次维护都留下可复用结果。
