# 多源转 Markdown

[[多源转Markdown]] 是把 PDF、Word、Excel、PPT、HTML、音频、YouTube 转录等复杂来源先转换为 Markdown，再交给 LLM Wiki 摄入的前置步骤。它让 [[持久化Wiki]] 能处理更多材料，同时保持文本可读、可链接、可版本化。

## 关联

- [[MarkItDown]] —— 当前 wiki 中主要的多格式转换工具。
- [[持久化Wiki]] —— 转换结果进入的长期维护层。
- [[Obsidian]] —— Markdown 是 Obsidian 的原生笔记格式。
