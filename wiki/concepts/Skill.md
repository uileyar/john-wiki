# Skill

[[Skill]] 是 agent 工作流中的复用层，适合保存会反复使用的方法、步骤和判断标准。它解决的是“同一类任务不要每次重新解释”的问题。

## 关联

- [[Agent工作流分工]] —— skill 是复用层。
- [[Command]] —— command 负责触发，skill 负责方法。
